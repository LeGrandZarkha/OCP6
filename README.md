OPENCLASSROOMS PROJET 6 -DEV WORDPRESS- Personnalisez un thème WordPress pour une startup avec HTML, CSS et PHP.

Le but de ce projet est de modifier un thème existant afin de correspondre à une maquette donnée. Il faut aussi intégrer un hook wp afin d'afficher le dashboard admin seulement si celui-ci est connecté.



!  Les fichiers style.css (responsive & correspondance maquette) du thème enfant et functions.php (hook wp) se trouvent dans wp-content/themes/Botiga-Child/  !
